Detailed Description

ModelWrapper is MATLAB object that (usually) wraps COMSOL objects. The COMSOL object is placed in the 'node' property. You can dynamically add properties to instances with addprop(H,'PropertyName') see http://www.mathworks.com/help/matlab/matlab_oop/dynamic-properties&ndash;adding-properties-to-an-instance.html#brffvja

Because this class is derived from dynamicprops, which is in turn derived from handle, you can pass it around and it will be passed by reference, not by value
